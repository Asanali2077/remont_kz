## Быстрая ориентация

```
app/                  # Next.js App Router (страницы + API)
  api/                # Backend — все route.ts файлы
  (страницы)/         # Frontend page.tsx файлы
components/           # React компоненты
lib/                  # Утилиты, типы, API-клиент
prisma/               # Схема БД и миграции
public/uploads/       # Загружаемые файлы (изображения, аудио)
```

## Бизнес-логика (коротко)

1. **Клиент** создаёт заявку (Request) — описание, категория, город, бюджет
2. **Компания** видит неназначенные заявки по своим категориям/городам
3. Компания делает оффер (цена + сообщение)
4. Клиент принимает оффер → заявка переходит к этой компании
5. Работа выполняется, статус меняется: `new → accepted → in_progress → completed`
6. Клиент оставляет рейтинг и отзыв
